## Description

Contactify app for Metasite interview

## Running the app

To run the app locally, clone this repo , open it, install and run.

```bash
# clone repo
git clone git@github.com:nudabagana/contactify.git

# open it
cd contactify
#
yarn install

# run dev
yarn dev

```
